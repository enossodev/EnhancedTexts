# EnhancedTexts

EnhancedTexts is a BetterDiscord plugin that improves Discord messages with AI before sending them.

It is designed for Discord announcements, support-ticket replies, direct messages, staff communication, moderation responses, application reviews, and community management messages.

## Features

- AI-powered message rewriting
- Discord Announcement mode
- Support Ticket mode
- Direct Message mode
- Logical Message React for replying with context
- [NEW] Additional Context system
- [NEW] Profile system for project-specific context
- [NEW] Language selection
- [NEW] Age Personalisation
- Token-based usage system
- Activation key validation
- Token balance display with usage bar
- Add Tokens modal
- Auto-copy option
- Auto-insert option
- Discord markdown support
- Emoji control
- BetterDiscord settings panel

## Requirements

- BetterDiscord
- A valid EnhancedTexts activation key
